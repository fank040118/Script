在这一节中，作者开始介绍如何求解一阶微分方程，而线性一阶微分方程是首先被讨论的一种特殊类型。
1. 线性一阶微分方程的形式
本节主要讨论的是形如以下标准形式的微分方程：
dydt+p(t)y=g(t)\frac{dy}{dt} + p(t)y = g(t)dtdy​+p(t)y=g(t)
其中，p(t)p(t)p(t) 和 g(t)g(t)g(t) 是给定区间上的连续函数。这种形式的微分方程被称为一阶线性微分方程。
2. 求解方法：积分因子法 (Integrating Factor Method)
求解这类微分方程的核心思想是引入一个特殊的函数，称为积分因子 μ(t)\mu(t)μ(t) (integrating factor)，它能将微分方程的左边转化为一个乘积的导数形式，从而简化求解过程。
整个推导过程如下：
1.
假设积分因子存在：首先，我们假设存在一个函数 μ(t)\mu(t)μ(t)。
2.
乘以积分因子：将原始微分方程的每一项都乘以 μ(t)\mu(t)μ(t)： μ(t)dydt+μ(t)p(t)y=μ(t)g(t)\mu(t) \frac{dy}{dt} + \mu(t)p(t)y = \mu(t)g(t)μ(t)dtdy​+μ(t)p(t)y=μ(t)g(t) [41, 2.2]
3.
关键假设：为了让左边成为一个乘积的导数，我们假设 μ(t)\mu(t)μ(t) 满足以下条件： μ(t)p(t)=μ′(t)\mu(t)p(t) = \mu'(t)μ(t)p(t)=μ′(t) [41, 2.3] 这个条件是积分因子法的“魔力”所在。
4.
应用乘积法则：将上述假设代入微分方程，得到： μ(t)dydt+μ′(t)y=μ(t)g(t)\mu(t) \frac{dy}{dt} + \mu'(t)y = \mu(t)g(t)μ(t)dtdy​+μ′(t)y=μ(t)g(t) [42, 2.4] 此时，方程的左边正好是乘积法则的逆运算，即 (μ(t)y(t))′(\mu(t)y(t))'(μ(t)y(t))′： (μ(t)y(t))′=μ(t)g(t)(\mu(t)y(t))' = \mu(t)g(t)(μ(t)y(t))′=μ(t)g(t) [42, 2.5]
5.
积分求解：现在，为了求得 y(t)y(t)y(t)，我们可以对等式两边进行积分： ∫(μ(t)y(t))′dt=∫μ(t)g(t)dt\int (\mu(t)y(t))' dt = \int \mu(t)g(t) dt∫(μ(t)y(t))′dt=∫μ(t)g(t)dt μ(t)y(t)+c1=∫μ(t)g(t)dt\mu(t)y(t) + c_1 = \int \mu(t)g(t) dtμ(t)y(t)+c1​=∫μ(t)g(t)dt 这里需要注意，积分常数 c1c_1c1​ 是至关重要的，不能遗漏。
6.
解出 y(t)y(t)y(t)：最后，通过简单的代数运算，我们可以解出 y(t)y(t)y(t)： μ(t)y(t)=∫μ(t)g(t)dt−c1\mu(t)y(t) = \int \mu(t)g(t) dt - c_1μ(t)y(t)=∫μ(t)g(t)dt−c1​ y(t)=∫μ(t)g(t)dt−c1μ(t)y(t) = \frac{\int \mu(t)g(t) dt - c_1}{\mu(t)}y(t)=μ(t)∫μ(t)g(t)dt−c1​​ 为了符号上的方便，通常我们会将积分常数前的负号吸收到常数 ccc 中，写作 y(t)=∫μ(t)g(t)dt+cμ(t)y(t) = \frac{\int \mu(t)g(t) dt + c}{\mu(t)}y(t)=μ(t)∫μ(t)g(t)dt+c​。
3. 积分因子的确定
既然已经有了 y(t)y(t)y(t) 的通用解形式，那么我们如何确定这个神奇的函数 μ(t)\mu(t)μ(t) 呢？这比您想象的要简单。 我们从关键假设 μ(t)p(t)=μ′(t)\mu(t)p(t) = \mu'(t)μ(t)p(t)=μ′(t) [41, 2.3] 开始。这是一个可分离变量的一阶微分方程： dμdt=p(t)μ\frac{d\mu}{dt} = p(t)\mudtdμ​=p(t)μ dμμ=p(t)dt\frac{d\mu}{\mu} = p(t)dtμdμ​=p(t)dt 对两边积分得到： ∫dμμ=∫p(t)dt\int \frac{d\mu}{\mu} = \int p(t)dt∫μdμ​=∫p(t)dt ln⁡∣μ(t)∣=∫p(t)dt+k\ln|\mu(t)| = \int p(t)dt + kln∣μ(t)∣=∫p(t)dt+k 其中 kkk 是积分常数。为了从自然对数中解出 μ(t)\mu(t)μ(t)，对两边取指数： ∣μ(t)∣=e∫p(t)dt+k=eke∫p(t)dt|\mu(t)| = e^{\int p(t)dt + k} = e^k e^{\int p(t)dt}∣μ(t)∣=e∫p(t)dt+k=eke∫p(t)dt 由于 eke^kek 也是一个未知常数，我们可以将其重新命名为 kkk，并且通常我们会选择 k=1k=1k=1 来得到最简单的积分因子形式，因为它最终会被吸收到 ccc 中。 因此，积分因子 μ(t)\mu(t)μ(t) 的公式为： μ(t)=e∫p(t)dt\mu(t) = e^{\int p(t)dt}μ(t)=e∫p(t)dt [49, 2.10] 最终的通解公式为： y(t)=∫μ(t)g(t)dt+cμ(t)y(t) = \frac{\int \mu(t)g(t) dt + c}{\mu(t)}y(t)=μ(t)∫μ(t)g(t)dt+c​ [49, 2.9]
4. 求解线性一阶微分方程的步骤总结
为了方便实际操作，作者总结了求解一阶线性微分方程的步骤：
1.
转化为标准形式：确保微分方程写为 dydt+p(t)y=g(t)\frac{dy}{dt} + p(t)y = g(t)dtdy​+p(t)y=g(t) 的形式。
2.
计算积分因子：使用公式 μ(t)=e∫p(t)dt\mu(t) = e^{\int p(t)dt}μ(t)=e∫p(t)dt 找到积分因子 μ(t)\mu(t)μ(t)。在计算指数上的积分时，通常可以省略积分常数，因为它最终会被其他常数吸收。
3.
乘以积分因子：将整个微分方程乘以 μ(t)\mu(t)μ(t)，并验证左边是否变为乘积的导数 (μ(t)y(t))′(\mu(t)y(t))'(μ(t)y(t))′。
4.
积分：对两边积分，并务必正确处理积分常数。
5.
解出 y(t)y(t)y(t)：通过代数运算解出 y(t)y(t)y(t)。
